# Artifact 
### meta requirements for the artifact 
- .zip file containing 
	- markdown file for core requirements
	- Screenshots and visual assets
		- these will be referenced in the main document by name

## Artifact content
### PRD
- Goals
- Constraints
- Use cases and user flows
- UI mocks or inspiration 
### Tech spec
- Arch diagram 
- tech stack 
	- with tradeoff decisions listed
- back end tasks 
- front end tasks
- test cases and criteria
- logging and ops requirements 
### Program 
- Milestones
	- Stopping points for user validation
