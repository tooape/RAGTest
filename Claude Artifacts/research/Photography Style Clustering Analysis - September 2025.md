# Photography [[Style Home|Style]] Clustering Analysis
---

## Overview

Analysis of 468 photographs (156 base images with 3 processing variations each) to identify distinct visual [[Style Home|style]] clusters based on technical and aesthetic characteristics using standardized photography [[Style Home|style]] tags.

## [[Style Home|Style]] Tag Categories Analyzed

The analysis used 129 standardized [[Style Home|style]] tags across three main categories:
- **[[Style Home|Color]] & White Balance**: [[Style Home|Color]] grading, saturation, temperature, tinting
- **Exposure & Contrast**: Highlights, shadows, blacks, whites, key levels
- **Processing Effects**: Grain, lighting, focus, and artistic treatments

## Processing Variation Pattern

Each base image has 3 consistent processing treatments:
- **Base (.jpg)**: Natural daylight processing
- **Variant 2 (-2.jpg)**: Enhanced warmth and contrast
- **Variant 3 (-3.jpg)**: Cool toning with lifted exposure

## Clustering Methodology

Images were grouped based on shared [[Style Home|style]] tag profiles rather than subject matter. The analysis revealed that:
- **Subject matter was ignored** as instructed
- **Visual [[Style Home|style]] consistency** determined groupings
- **Processing variations** created sub-clusters within each [[Style Home|style]] family
- **Technical characteristics** (grain, contrast, [[Style Home|color]]) were primary clustering factors

## Key Findings

1. **6 distinct visual [[Style Home|style]] families** identified across the dataset
2. **Consistent processing patterns** across different subjects
3. **[[Style Home|Style]] variations independent of content** - same processing applied regardless of scene type
4. **Technical characteristics dominate** over aesthetic content in clustering

## Recommendations

For [[Style Home|style]]-based image organization:
- Use technical [[Style Home|style]] tags as primary clustering criteria
- Group by processing characteristics rather than visual content
- Consider white balance and contrast as primary differentiators
- Account for grain and [[Style Home|color]] saturation in [[Style Home|style]] classification
