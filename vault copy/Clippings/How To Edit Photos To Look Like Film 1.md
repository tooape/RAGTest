# [How To Edit Photos To Look Like Film](https://alikgriffin.com/how-to-edit-like-film/)

[[November 19, 2025]] 


## Summary
Learn to edit digital photos like film. Use diffusion filters for a soft glow, adjust curves for tonality (soft highlights, crunchy/lifted shadows), apply color shifts (cyan highlights, magenta shadows) via RGB curves/Calibration, and add film grain for texture.
