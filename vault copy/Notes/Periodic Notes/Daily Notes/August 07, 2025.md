### Current [[Photoshop]] contextual toolbar
Current Photoshop offers users a contextual toolbar based on the current selected tool. The options and their display order are currently rule based, with no active intelligence. Could this be replaced with a learned system and could that system provide value to a creator?

### Offline learning approach 
Could we track this behavior and train something offline? 

### SDC / [[Photoshop]] value
There are some key product viability questions to answer: 
1. Tools are a much smaller set of options than media in stock, limiting value in contextualizing these options. 
2. Creators tend to be much more conservative than communicators, creating more natural resistance to this kind of feature. 
3. Would a learned approach provide substantial incremental value over the current approach? 
4. Could we actually get [[Photoshop]] to instrument their app and gather the mass of data required to train something effective here? 
	1. Using synthetic data would effectively be replicating the current implementation of rules. Rendering the project moot.
